# MagiesTerminal Website

MagiesTerminal 官方落地页（域名 [shell.magies.top](https://shell.magies.top)，桌面客户端仓库 [MgTerminal](https://github.com/JasonZhangDad/MgTerminal)）。

## 开发

```bash
npm install
npm run dev
```

本地地址：http://localhost:5174

## 构建

```bash
npm run build
npm run preview
```

产物输出到 `dist/`，可直接部署到静态托管。
